# 五子棋AI项目

一个基于Python实现的五子棋游戏，包含图形用户界面(GUI)和智能AI对战功能。

## 项目结构

gomoku/

├── main.py              # 程序入口

├── gui.py              # 图形用户界面

├── core.py             # 游戏核心逻辑

├── AI.py               # 高级AI实现

├── AIcheck.py          # 棋型检测模块

├── AIevaluate.py       # 评估函数模块


## 运行环境

- Python 3.7+
- 不需要额外安装包，使用标准库

## 安装与运行

### 直接运行

bash

python main.py

## 游戏功能

1. **双人对战模式**：两个玩家轮流下棋
2. **人机对战模式**：玩家与AI对战
3. **游戏控制**：
   - 重新开始：R键
   - 悔棋：Z键
   - 退出：ESC键
4. **可视化界面**：直观的棋盘和棋子显示

## AI功能

- 实现Minimax搜索算法
- 使用Alpha-Beta剪枝优化
- 包含VCF（冲四活三）必胜棋搜索
- 智能局面评估
- 搜索深度可配置（默认4层）

## 开发者说明

如需修改AI参数，可在`AI.py`中调整：
- `depth`：搜索深度
- `vcf_depth`：VCF搜索深度
